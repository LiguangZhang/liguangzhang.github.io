Parser for signed configuration.

Signed configuration is stored an JSON inside APK meta-data. The format
of the JSON is as follows:
{
  "version": n,
  "config": [
    {
      "minSdk": n,
      "maxSdk": n,
      "values": [
        {
          "key": "global settings key",
          "value": "value for key"
        },
    ...
      ],
      ...
    }
  ]
}
